# AgentMergeGuard Product Brief

## One-liner

Before you merge AI-written code, run one guard.

## Product

AgentMergeGuard is a lightweight pre-merge risk guard for AI-generated PRs, MCP configs, AGENTS.md / CLAUDE.md / Cursor rules, GitHub Actions workflows, package scripts, and secret-like leaks.

## Target users

Primary:
- open-source maintainers
- small development teams using AI coding agents
- solo developers using Codex, Claude Code, Cursor, or Copilot
- agencies accepting AI-generated code
- teams experimenting with MCP servers

## Problem

AI coding agents can create code faster than maintainers can safely review it.

The new bottleneck is not writing code.
The bottleneck is deciding whether AI-written code, MCP configs, and agent instructions are safe to merge.
